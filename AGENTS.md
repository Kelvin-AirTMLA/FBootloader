# Agent instructions

## Scope

Do only what the user asks. Do not go beyond the request.

- If the user asks a question, **explain only** — do not edit files, run builds, refactor code, or add features unless they ask for that.
- If the user asks for one file or one change, do that one thing — do not "improve" surrounding code or add extras.
- Do not proactively create commits, PRs, tests, or documentation unless explicitly requested.

When in doubt, answer the question or make the single change requested — nothing more.
